Réponses : 1.1 Sélecteurs

  1. Tous les liens dans la nav : .main-nav a
  2. Lien actif : .main-nav a.active
  3. Article featured : article.featured
  4. Premier paragraphe de chaque article : article p:first-of-type
  5. Éléments li pairs : li:nth-child(even)

Réponses : 1.2 Spécificité

  1. Classement croissant :
    p (0,0,1) → .text (0,1,0) → p.text (0,1,1) → #main (1,0,0) → #main .text p (1,1,1)
  2. Même spécificité : La règle déclarée en dernier dans le CSS gagne (ordre de cascade).
  3. Pourquoi éviter !important : Il casse la cascade naturelle du CSS, rend le code difficile à déboguer et à maintenir.
