Exercice 2 : Box Model

2.1 – Visualisation du Box Model

content-box
width: 300px, padding: 20px, border: 5px
→ Largeur totale visible = 300 + 20×2 + 5×2 = 350px
border-box
width: 300px, padding: 20px, border: 5px
→ Largeur totale visible = 300px (tout inclus)

Réponses :

  1. content-box : largeur totale = 300 + (20×2) + (5×2) = 350px
  2. border-box : largeur totale = 300px (padding et border sont inclus)
  3. Reset CSS universel : 
    *, *::before, *::after {
    box-sizing: border-box;
}
