Exercice 6 : Variables CSS et thèmes

Carte d'exemple

Ce texte utilise des variables CSS pour ses couleurs, espacements et tailles de police. Cliquez 🌙 pour basculer le thème !

Système de variables CSS complet

Les variables sont regroupées par catégorie :

/* ── Couleurs ── */
--color-bg        : #f9f9f9;
--color-surface   : #ffffff;
--color-text      : #222222;
--color-primary   : #007bff;
--color-secondary : #6c757d;

/* ── Typographie ── */
--font-base  : 16px;
--font-lg    : 20px;
--font-sm    : 13px;
--font-title : 2rem;

/* ── Espacement ── */
--space-sm : 8px;
--space-md : 16px;
--space-lg : 32px;

/* ── Apparence ── */
--radius : 10px;
--shadow : 0 4px 12px rgba(0,0,0,0.08);

Utilisation avec var()

.card {
    background : var(--color-surface);
    padding    : var(--space-md);
    border-radius : var(--radius);
    font-size  : var(--font-base);
    box-shadow : var(--shadow);
}

h1 {
    font-size : var(--font-title);
    color     : var(--color-primary);
}

En mode sombre, on surcharge les variables :

[data-theme="dark"] {
    --color-bg      : #0f1117;
    --color-surface : #1a1d27;
    --color-text    : #e0e0e0;
    --color-primary : #4da6ff;
}
