Exercice 8 : Quiz de révision CSS3

1. Quel sélecteur a la spécificité la plus haute ?

Réponse correcte : b) #header a
Un ID (#) vaut 1,0,0 → (1,0,1) au total.
a) .header .nav a = (0,2,1) — b) #header a = (1,0,1)
c) header nav a = (0,0,3) — d) a.active = (0,1,1)

2. Avec box-sizing: border-box, un élément avec width: 200px et padding: 20px a une largeur totale de :

Réponse correcte : b) 200px
Avec border-box, le padding est INCLUS dans la width déclarée. La largeur totale visible = 200px. Le contenu interne fait 200 - 20×2 = 160px.

3. Pour centrer un élément verticalement et horizontalement avec Flexbox :

Réponse correcte : c)
justify-content: center = centrage sur l'axe principal (horizontal par défaut)
align-items: center = centrage sur l'axe secondaire (vertical par défaut)
Les deux ensemble = centrage dans les deux directions.

4. La valeur 1fr en CSS Grid signifie :

Réponse correcte : b)
fr = fraction. 1fr 2fr = l'espace est divisé en 3 parts : la 1ère colonne en prend 1, la 2ème en prend 2.

5. L'approche "Mobile First" utilise :

Réponse correcte : b) min-width
Mobile First = on écrit d'abord le CSS pour mobile (sans media query),
puis on utilise @media (min-width: 768px) pour ajouter les styles tablette/desktop.
